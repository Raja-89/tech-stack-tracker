A modern web application designed to help developers track, manage, and visualize their technology skills and learning progress. Built with performance in mind using **Vite** and styled with **Tailwind CSS**.

## üöÄ Features

* **Tech Tracking:** Log and organize languages, frameworks, and tools you are learning or mastering.
* **Modern UI:** Clean and responsive interface built with Tailwind CSS.
* **Fast Development:** Powered by Vite for lightning-fast hot module replacement (HMR).
* **Responsive Design:** Optimized for both desktop and mobile devices.
